## Method

- Treated "line by line" as **claim by claim**: each checkable factual assertion (numbers, measurable outcomes, historical claims) is assessed for accuracy.
- Did **not** score: value judgments ("best ever"), vague boasts, promises, proposals, or predictions.
- **Verdict labels:** TRUE, FALSE, PARTLY TRUE, MISLEADING, UNSUPPORTED/UNVERIFIABLE.
- Sources used include nonpartisan and primary data (e.g., BLS/BEA/CBP/EIA/Freddie Mac) as synthesized by independent fact-checks. [R5] [R6]
